# 프론트 TODO (개발 B)

> 목표: 채팅 화면에서 말/글로 입력하면 백엔드 답이 말풍선으로 뜬다.
> 스택: React + Vite. 음성은 브라우저 Web Speech API.
> 공통 규칙/접점 계약: [README.md](./README.md) 먼저 읽기.

## 실행법
```bash
cd frontend
npm install
npm run dev        # http://localhost:5173  (백엔드도 8000 에 같이 띄워두기)
```

## 현재 뼈대 상태 (이미 되는 것)
- 메시지 목록 렌더 + 입력창/전송 버튼 (`src/App.jsx`)
- 백엔드 호출 (`src/api.js` — `POST /chat` → `reply`)
- 최소 스타일 (`src/styles.css`)
- → 백엔드가 스텁이어도 "받은 말: ..." 응답이 말풍선에 뜬다. **먼저 이게 뜨는지 확인.**
