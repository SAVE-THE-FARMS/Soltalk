## 반달허브 (VandalHub) — 곤충 스마트팜 원격제어 앱

- 링크: https://vandalsoft.com/ko-vandalhub-introduction/
- 용도: **대시보드 인터페이스 참고용**

(주)반달소프트의 곤충 스마트팜 원격제어 애플리케이션. **중년층도 쉽게 쓰도록 설계**된
점이 SolTalk 의 고령 농가 사용자 타깃과 방향이 같다.

화면 구성 (4개):
1. **장치 제어·편집** — 환풍기/가습기/냉·난방기 등을 자동·수동 제어
2. **가동 설정** — 온·습도, 급수 시간 등 사육 환경 설정
3. **온·습도 데이터 열람** — 주간·일간 데이터를 그래프로 확인
4. **기타 관리** — 장치 편집·설정 페이지

모바일 우선 + PC/태블릿 버전 제공 → SolTalk 의 모바일 우선 반응형 방침과 동일.
