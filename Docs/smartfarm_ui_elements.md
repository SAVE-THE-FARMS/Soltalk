# 스마트팜 AI 에이전트 — 1차 UI 요소 문서

> 대상: 자연어 제어·조회 + 대시보드 + 인앱 알림 (음성/카카오톡 연동/솔캐스트 예측은 2차 확장)

---

## 2.1 채팅 화면

- 말풍선 UI (사용자/AI 구분, 카카오톡 스타일)
- 하단 고정 입력창 + 전송 버튼
- AI 응답 내 액션 결과는 일반 텍스트와 다르게 강조 표시 (예: 연한 배경색 카드로 "✅ 2번 온실 창문 열림" 표시)
- 로딩 인디케이터 (Claude 응답 대기 중 "..." 표시)

---

## 2.2 대시보드 메인 화면

- **상단 알림 영역**: critical/warning 알림이 있을 때만 카드로 노출 (없으면 "모든 온실 정상입니다 ✅" 한 줄)
  - 카드 색상: 🔴 critical / 🟡 warning
  - 카드 클릭 → 상세 화면 이동
- **온실별 요약 카드 (그리드/리스트)**
  - 온실명, 상태 색상 배지(신호등 색), 온도/습도 숫자, 장비 상태 아이콘(차광막/창문/관수)
  - 정상 온실은 축소된 형태로, 문제 있는 온실은 상단 고정
- **생산량 요약 위젯**
  - 오늘/이번주 생산량 + 평균 대비 증감 화살표(▲▼)
- **오늘 처리 이력 리스트**
  - 시간순 타임라인, 자동/수동 구분 아이콘
- **리셋 버튼** (우측 상단, 데모 편의용, 눈에 띄지 않는 위치)

---

## 2.3 온실 상세 화면 (카드 클릭 시)

- 상단: 상태 배지 + 한 줄 원인 설명 ("습도 82%, 임계값 80% 초과 2일 지속")
- 중단: 추천 조치 + 원터치 실행 버튼 (예: [창문 열기] 버튼)
- 하단: 상태 변화 타임라인/그래프 (최근 며칠간 온도·습도 추이, 언제부터 이상이었는지 시각화)
- 뒤로가기로 대시보드 복귀

---

## 2.4 알림(인앱 시뮬레이션) 컴포넌트

- 화면 상단 토스트/배너 형태 (실제 모바일 푸시는 데모 범위 밖이므로 웹 화면 내 시뮬레이션)
- 심각도별 스타일 차등
  - 🔴 critical: 화면 상단 강제 노출 배너 + (가능하면) 알림음 재생 + 원터치 액션 버튼 내장
  - 🟡 warning: 우측 상단 배지 카운트만 증가, 클릭 시 알림함 노출
- 알림함(리스트) UI: 알림 목록 + 각 항목별 액션 버튼 + 닫기(dismiss) 버튼
- 노랑→빨강 승격 시 배너 스타일이 실시간으로 전환되는 효과 (데모 임팩트 포인트)

---

## 2.5 공통 컴포넌트

- 색상 토큰: critical(빨강) / warning(노랑) / normal(초록 또는 파랑) — 전 화면 통일
- 장비 상태 아이콘 세트 (차광막/창문/관수 각각 on/off/partial 상태 아이콘)
- 상단 네비게이션: 채팅 화면 ↔ 대시보드 화면 전환 탭 또는 버튼
